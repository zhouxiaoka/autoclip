# ❓ AutoClip Desktop 常见问题解答

## 🚀 安装和启动

### Q: 应用无法启动怎么办？

**A:** 请按以下步骤排查：

1. **检查系统要求**
   - macOS: 10.13+ (Intel/Apple Silicon)
   - Windows: Windows 10+ (64位)
   - Linux: Ubuntu 18.04+ 或其他主流发行版

2. **重新安装应用**
   - 完全卸载旧版本
   - 重新下载最新安装包
   - 按照安装指南重新安装

3. **检查系统权限**
   - macOS: 系统偏好设置 > 安全性与隐私 > 允许应用运行
   - Windows: 以管理员身份运行
   - Linux: 检查文件执行权限

4. **查看错误日志**
   - 检查系统日志中的错误信息
   - 联系技术支持并提供错误详情

### Q: 首次运行时提示"无法验证开发者"？

**A:** 这是macOS的安全机制，解决方法：

1. 打开 **系统偏好设置** > **安全性与隐私**
2. 在 **通用** 标签页中找到被阻止的应用
3. 点击 **"仍要打开"** 按钮
4. 确认允许应用运行

### Q: 应用启动后显示空白页面？

**A:** 可能的原因和解决方案：

1. **网络连接问题**
   - 检查网络连接
   - 尝试刷新页面 (F5)

2. **端口冲突**
   - 重启应用
   - 检查8000端口是否被占用

3. **缓存问题**
   - 清除应用缓存
   - 重启应用

## 📁 文件上传

### Q: 支持哪些视频格式？

**A:** 支持的视频格式：
- **MP4** (推荐)
- **AVI**
- **MOV**
- **MKV**
- **WEBM**
- **FLV**
- **WMV**

### Q: 文件大小有限制吗？

**A:** 是的，根据性能设置：

- **低性能模式**: 最大512MB
- **中等性能模式**: 最大2GB (推荐)
- **高性能模式**: 最大5GB

### Q: 上传失败怎么办？

**A:** 请检查以下项目：

1. **文件格式**
   - 确保使用支持的视频格式
   - 推荐使用MP4格式

2. **文件大小**
   - 检查文件是否超过大小限制
   - 可以压缩视频文件

3. **网络连接**
   - 检查网络连接是否稳定
   - 尝试重新上传

4. **磁盘空间**
   - 确保有足够的磁盘空间
   - 至少需要文件大小的2倍空间

### Q: 可以上传字幕文件吗？

**A:** 可以，支持的字幕格式：
- **SRT** (推荐)
- **VTT**
- **ASS**
- **SSA**

上传字幕文件可以：
- 提高语音识别的准确性
- 加快处理速度
- 提供更精确的时间轴

## 🤖 AI配置

### Q: 如何配置语音识别？

**A:** 有两种方式：

#### 本地Whisper模型
1. 进入 **设置** > **语音转写配置**
2. 选择 **"本地Whisper模型"**
3. 选择模型大小：
   - `base`: 平衡速度和准确性 (推荐)
   - `small`: 快速处理
   - `medium`: 高准确性
   - `large`: 最高准确性
4. 点击 **"下载模型"** (首次使用)

#### 云端API服务
1. 选择 **"云端API服务"**
2. 选择服务提供商：
   - OpenAI Whisper API
   - Azure Speech Services
   - Google Cloud Speech-to-Text
   - 阿里云语音识别
3. 输入API密钥
4. 测试连接

### Q: 如何获取API密钥？

**A:** 各服务商获取方式：

#### 阿里通义千问
1. 访问 [阿里云控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录账号
3. 开通通义千问服务
4. 获取API密钥

#### OpenAI
1. 访问 [OpenAI官网](https://platform.openai.com/)
2. 注册/登录账号
3. 创建API密钥
4. 充值账户余额

#### Google Gemini
1. 访问 [Google AI Studio](https://makersuite.google.com/)
2. 使用Google账号登录
3. 获取API密钥
4. 免费额度：每分钟15次请求

### Q: API连接测试失败？

**A:** 请检查：

1. **API密钥**
   - 确保密钥正确
   - 检查是否过期
   - 确认有足够余额

2. **网络连接**
   - 检查网络是否正常
   - 确认可以访问API服务

3. **服务状态**
   - 检查API服务是否正常
   - 查看服务商状态页面

4. **配置信息**
   - 确认所有配置项正确
   - 检查地区设置

## ⚙️ 性能设置

### Q: 如何选择合适的性能级别？

**A:** 根据设备配置选择：

#### 低性能模式
- **适用**: 老旧设备、资源受限
- **限制**: 文件512MB、1个并发任务
- **内存**: 2GB

#### 中等性能模式 (推荐)
- **适用**: 普通用户、日常使用
- **限制**: 文件2GB、2个并发任务
- **内存**: 4GB

#### 高性能模式
- **适用**: 专业用户、高配置设备
- **限制**: 文件5GB、4个并发任务
- **内存**: 8GB

### Q: 处理速度很慢怎么办？

**A:** 优化建议：

1. **提高性能设置**
   - 升级到更高的性能级别
   - 增加并发任务数

2. **使用云端API**
   - 本地处理较慢
   - 云端API速度更快

3. **选择合适模型**
   - 使用较小的Whisper模型
   - 平衡速度和准确性

4. **关闭其他应用**
   - 释放系统资源
   - 提高处理速度

### Q: 内存不足怎么办？

**A:** 解决方案：

1. **调整性能设置**
   - 降低到低性能模式
   - 减少并发任务数

2. **关闭其他应用**
   - 释放内存资源
   - 重启应用

3. **清理系统缓存**
   - 清理临时文件
   - 重启系统

## 🎬 视频处理

### Q: 处理时间需要多久？

**A:** 处理时间取决于：

1. **视频长度**
   - 1小时视频约需10-30分钟
   - 具体时间因设备而异

2. **视频质量**
   - 高分辨率视频处理更慢
   - 音频质量影响语音识别

3. **设备性能**
   - CPU性能影响处理速度
   - 内存大小影响并发能力

4. **AI服务选择**
   - 云端API通常更快
   - 本地处理需要更多时间

### Q: 切片质量评分低怎么办？

**A:** 优化建议：

1. **视频质量**
   - 提高视频分辨率
   - 改善音频质量
   - 减少背景噪音

2. **内容优化**
   - 语速适中
   - 表达清晰
   - 避免长停顿

3. **调整设置**
   - 降低评分阈值
   - 调整切片参数
   - 使用更准确的AI模型

### Q: 可以手动调整切片吗？

**A:** 可以，支持以下操作：

1. **编辑切片信息**
   - 修改标题和描述
   - 调整时间范围

2. **删除不需要的切片**
   - 移除低质量切片
   - 清理重复内容

3. **合并相关切片**
   - 将相关切片合并
   - 创建更完整的内容

## 📚 合集管理

### Q: 如何创建合集？

**A:** 创建步骤：

1. **选择切片**
   - 在项目详情页面选择切片
   - 可以按评分、时间等筛选

2. **创建合集**
   - 点击"创建合集"按钮
   - 输入合集标题和描述

3. **选择类型**
   - AI推荐：系统自动推荐
   - 手动创建：用户手动选择

4. **完成创建**
   - 确认合集信息
   - 点击"创建合集"

### Q: 合集可以导出吗？

**A:** 可以，支持多种导出方式：

1. **视频文件**
   - 导出为MP4格式
   - 包含所有切片内容

2. **分享链接**
   - 生成在线分享链接
   - 支持二维码分享

3. **数据导出**
   - 导出合集信息为JSON
   - 导出切片列表为CSV

### Q: 如何管理多个合集？

**A:** 管理建议：

1. **分类组织**
   - 按主题分类
   - 按时间排序
   - 使用标签标记

2. **定期整理**
   - 删除不需要的合集
   - 更新合集内容
   - 优化合集结构

3. **备份重要合集**
   - 导出重要合集
   - 定期备份数据
