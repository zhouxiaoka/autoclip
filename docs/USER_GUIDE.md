# 📚 AutoClip Desktop 用户使用指南

## 🎯 快速开始

### 第一次使用

1. **启动应用**：双击桌面图标或从应用程序文件夹启动
2. **首次配置向导**：系统会自动引导您完成初始设置
3. **配置AI服务**：设置语音识别和AI模型API密钥
4. **开始使用**：上传视频文件，开始您的第一个项目

### 基本工作流程

```
上传视频 → 语音转写 → AI分析 → 生成切片 → 创建合集 → 导出分享
```

## 📁 项目管理

### 创建新项目

1. 点击首页的 **"新建项目"** 按钮
2. 选择视频文件（支持MP4、AVI、MOV、MKV、WEBM格式）
3. 可选：上传字幕文件（SRT、VTT格式）
4. 输入项目名称和描述
5. 选择视频分类（如：教育、娱乐、商业等）
6. 点击 **"开始处理"**

### 项目状态说明

- **🔄 处理中**：视频正在分析处理
- **✅ 已完成**：处理完成，可以查看结果
- **❌ 失败**：处理失败，请检查错误信息
- **⏸️ 暂停**：处理已暂停

### 项目操作

- **查看详情**：点击项目卡片查看详细信息
- **重新处理**：失败的项目可以重新开始处理
- **删除项目**：删除不需要的项目和文件
- **导出项目**：导出项目数据和结果

## 🎬 视频切片管理

### 查看切片

1. 进入项目详情页面
2. 在 **"切片列表"** 中查看所有生成的切片
3. 每个切片显示：
   - 标题和描述
   - 时间范围
   - 质量评分
   - 预览缩略图

### 切片操作

- **播放预览**：点击切片卡片播放视频片段
- **编辑信息**：修改切片标题和描述
- **调整时间**：微调切片的开始和结束时间
- **删除切片**：移除不需要的切片

### 切片质量评分

系统会根据以下标准为每个切片评分：

- **内容质量**：信息密度、逻辑清晰度
- **语言表达**：语速适中、表达清晰
- **视觉质量**：画面稳定、光线充足
- **时长合适**：不会太短或太长

评分范围：0-100分，推荐使用70分以上的切片。

## 📚 合集管理

### 创建合集

1. 在项目详情页面，点击 **"创建合集"**
2. 选择要包含的切片
3. 输入合集标题和描述
4. 选择合集类型：
   - **AI推荐**：系统自动推荐的相关切片
   - **手动创建**：用户手动选择的切片
5. 点击 **"创建合集"**

### 合集操作

- **查看合集**：浏览合集内容和切片列表
- **编辑合集**：修改标题、描述和包含的切片
- **重新排序**：调整切片在合集中的顺序
- **导出合集**：生成完整的视频文件
- **分享合集**：获取分享链接或二维码

### 合集类型

- **主题合集**：围绕特定主题的切片集合
- **时间合集**：按时间顺序组织的切片
- **质量合集**：高质量切片的精选集合
- **自定义合集**：用户自定义的切片组合

## ⚙️ 系统设置

### AI模型配置

#### 语音识别设置

**本地Whisper模型**
- 优点：免费、隐私安全、离线可用
- 缺点：需要下载模型文件、处理速度较慢
- 推荐模型：
  - `base`：平衡速度和准确性
  - `small`：较快速度，适合实时处理
  - `medium`：较高准确性，适合重要内容
  - `large`：最高准确性，适合专业用途

**云端API服务**
- 优点：处理速度快、准确性高、无需本地资源
- 缺点：需要网络连接、可能产生费用
- 支持服务：
  - OpenAI Whisper API
  - Azure Speech Services
  - Google Cloud Speech-to-Text
  - 阿里云语音识别

#### 大语言模型设置

**推荐配置**
- **阿里通义千问**：中文支持好，性价比高
- **OpenAI GPT**：功能强大，支持多语言
- **Google Gemini**：免费额度大，响应快速
- **硅基流动**：国内服务，访问稳定

### 性能设置

#### 性能级别

- **低性能模式**：适合资源受限的设备
  - 文件大小限制：512MB
  - 并发处理：1个任务
  - 内存使用：2GB
  
- **中等性能模式**：平衡性能和资源使用（推荐）
  - 文件大小限制：2GB
  - 并发处理：2个任务
  - 内存使用：4GB
  
- **高性能模式**：适合资源充足的设备
  - 文件大小限制：5GB
  - 并发处理：4个任务
  - 内存使用：8GB

#### 存储设置

- **数据目录**：设置项目文件存储位置
- **缓存管理**：清理临时文件和缓存
- **自动清理**：定期清理旧项目文件

### 应用设置

#### 自动启动

- 启用后，应用会在系统启动时自动运行
- 可以通过系统托盘图标访问应用
- 适合需要频繁使用的用户

#### 通知设置

- 处理完成通知
- 错误提醒通知
- 系统状态通知

## 🚨 故障排除

### 常见问题

#### 应用无法启动

**症状**：双击应用图标后没有反应

**解决方案**：
1. 检查系统要求是否满足
2. 重新下载并安装应用
3. 检查系统权限设置
4. 查看系统日志中的错误信息

#### 视频上传失败

**症状**：上传视频时提示错误

**可能原因**：
- 文件格式不支持
- 文件大小超过限制
- 网络连接问题
- 磁盘空间不足

**解决方案**：
1. 检查文件格式和大小
2. 检查网络连接
3. 清理磁盘空间
4. 尝试重新上传

#### 处理速度慢

**症状**：视频处理时间过长

**优化建议**：
1. 调整性能设置到更高级别
2. 关闭其他占用资源的应用
3. 使用云端API服务
4. 选择较小的Whisper模型

#### API连接失败

**症状**：AI服务无法正常工作

**检查步骤**：
1. 验证API密钥是否正确
2. 检查网络连接
3. 确认API服务状态
4. 检查账户余额或配额

### 错误代码说明

| 错误代码 | 说明 | 解决方案 |
|---------|------|----------|
| ERR_001 | 文件格式不支持 | 使用支持的视频格式 |
| ERR_002 | 文件大小超限 | 压缩视频或使用分片上传 |
| ERR_003 | API密钥无效 | 检查并更新API密钥 |
| ERR_004 | 网络连接失败 | 检查网络连接 |
| ERR_005 | 磁盘空间不足 | 清理磁盘空间 |
| ERR_006 | 内存不足 | 关闭其他应用或调整设置 |

## 📊 最佳实践

### 视频准备建议

1. **视频质量**：
   - 分辨率：1080p或以上
   - 帧率：30fps或以上
   - 音频：清晰无杂音

2. **内容结构**：
   - 主题明确，逻辑清晰
   - 避免过长的停顿
   - 语速适中，表达清晰

3. **文件格式**：
   - 推荐使用MP4格式
   - 文件大小控制在2GB以内
   - 确保视频可以正常播放

### 切片优化技巧

1. **标题优化**：
   - 简洁明了，突出要点
   - 包含关键词，便于搜索
   - 长度控制在30字以内

2. **描述完善**：
   - 详细描述切片内容
   - 包含关键信息点
   - 便于后续查找和使用

3. **质量筛选**：
   - 优先使用高评分切片
   - 定期清理低质量内容
   - 建立质量评估标准

### 合集组织策略

1. **主题分类**：
   - 按内容主题组织
   - 建立清晰的分类体系
   - 便于内容管理和查找

2. **时间管理**：
   - 定期整理和更新合集
   - 标记重要和常用合集
   - 建立内容更新计划

3. **分享优化**：
   - 为合集添加吸引人的标题
   - 编写详细的合集描述
   - 选择合适的分享时机
