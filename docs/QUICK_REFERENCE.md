# ⚡ AutoClip Desktop 快速参考

## 🚀 快速开始

### 第一次使用
1. 启动应用 → 首次配置向导 → 设置API密钥 → 开始使用

### 基本流程
```
上传视频 → 等待处理 → 查看切片 → 创建合集 → 导出分享
```

## 📋 快捷键

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 新建项目 | `Ctrl/Cmd + N` | 快速创建新项目 |

## 🔧 常见操作

### 项目管理

#### 创建项目
1. 点击 "新建项目"
2. 选择视频文件
3. 输入项目信息
4. 点击 "开始处理"

#### 项目状态
- 🔄 **处理中**: 正在分析视频
- ✅ **已完成**: 可以查看结果
- ❌ **失败**: 需要重新处理
- ⏸️ **暂停**: 处理已暂停

### 切片管理

#### 切片评分标准
- **90-100分**: 优秀，强烈推荐
- **80-89分**: 良好，推荐使用
- **70-79分**: 一般，可以使用
- **60-69分**: 较差，谨慎使用
- **60分以下**: 不推荐使用

#### 切片操作
- **播放**: 点击切片卡片
- **编辑**: 点击编辑按钮
- **删除**: 点击删除按钮
- **添加到合集**: 选择后点击"创建合集"

### 合集管理

#### 合集类型
- **AI推荐**: 系统自动推荐
- **手动创建**: 用户手动选择
- **主题合集**: 按主题组织
- **时间合集**: 按时间顺序

#### 合集操作
- **查看**: 点击合集卡片
- **编辑**: 修改标题和描述
- **重新排序**: 拖拽调整顺序
- **导出**: 生成完整视频

## 🚨 故障排除速查

### 常见问题

#### 应用无法启动
```bash
检查项目:
1. 系统要求是否满足
2. 重新安装应用
3. 检查权限设置
4. 查看错误日志
```

#### 上传失败
```bash
可能原因:
- 文件格式不支持
- 文件大小超限
- 网络连接问题
- 磁盘空间不足

解决方案:
1. 检查文件格式和大小
2. 检查网络连接
3. 清理磁盘空间
4. 重新上传
```

#### 处理速度慢
```bash
优化建议:
1. 提高性能设置级别
2. 关闭其他应用
3. 使用云端API
4. 选择较小模型
```

#### API连接失败
```bash
检查步骤:
1. 验证API密钥
2. 检查网络连接
3. 确认服务状态
4. 检查账户余额
```

### 错误代码速查

| 代码 | 问题 | 解决 |
|------|------|------|
| ERR_001 | 文件格式不支持 | 使用MP4格式 |
| ERR_002 | 文件大小超限 | 压缩视频文件 |
| ERR_003 | API密钥无效 | 更新API密钥 |
| ERR_004 | 网络连接失败 | 检查网络设置 |
| ERR_005 | 磁盘空间不足 | 清理磁盘空间 |
| ERR_006 | 内存不足 | 调整性能设置 |
